# 📅 Reservation Calendar (예약 관리 달력)
*한 화면에서 Airbnb, Booking.com, 직접 예약 등을 통합 관리할 수 있는 웹 기반 숙소 예약 캘린더*

:contentReference[oaicite:0]{index=0}

## ✨ 프로젝트 소개
이 프로젝트는 숙박업 운영자가 **객실별 예약 현황**, **체크인·체크아웃**, **예약 출처**, **객실 상태(청소/숙박/비어있음)** 등을  
직관적인 UI로 관리할 수 있도록 제작된 **웹 앱 형태의 예약 관리 시스템**입니다.

TailwindCSS 기반으로 만들어져 가볍게 동작하며, 로컬 스토리지를 이용해 데이터가 유지됩니다.

---

## 🎯 주요 기능

### 1. 월별 예약 캘린더
- Airbnb / Booking.com / 직접 예약 등 **출처별 색상 구분**
- 날짜별 예약 바(Bar) 표시
- 객실별 필터링 기능
- 월 이동(이전/다음)

### 2. 예약 CRUD 기능
- 예약 추가(Create)
- 예약 상세 보기(Read)
- 예약 삭제(Delete)
- 예약 메모 저장 가능

### 3. 객실 상태 대시보드
- 객실별 상태 표시 (🟢비어있음 / 🔴숙박중 / 🟡청소중)
- 상태 클릭 시 모달에서 상세 편집 가능  
  - 현재 손님명  
  - 체크인 / 체크아웃 시간  
  - 객실 상태 저장  
- 상태는 로컬스토리지에 자동 저장됨

### 4. Booking.com API 연동(데모 모드 포함)
- API Key + Property ID 저장 가능
- 연결 테스트 기능
- Booking.com 예약을 시스템 형식으로 변환하여 자동 추가
- 실제 API 불가 시 데모 데이터 자동 추가

### 5. 반응형 UI (TailwindCSS)
- 모바일 브라우저에서도 사용 가능
- 모달 기반 인터페이스로 조작이 편리함

## 📂 프로젝트 구조
```
reservation-calendar.html
├─ 헤더 및 설명
├─ API 설정 모달
├─ 예약 추가 모달
├─ 예약 상세 모달
├─ 객실 상태 모달
├─ 캘린더 렌더링 스크립트
└─ 예약/객실 상태 관리 로직
```

---

## 🚀 사용 방법

1. `reservation-calendar.html` 파일을 브라우저에서 열기만 하면 실행됩니다.
2. 상단에서 **월 이동 / 객실 필터링 / 예약 추가** 기능을 사용할 수 있습니다.
3. "설정" 버튼을 눌러 **Booking.com API 키와 Property ID**를 저장할 수 있습니다.
4. 객실 상태 카드를 눌러 객실 상태(청소/빈방/숙박)를 업데이트할 수 있습니다.
